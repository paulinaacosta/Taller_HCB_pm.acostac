Machine Learning

Informacion

Machine Learning es un conjunto de tecnicas estadisticas que permiten a un programa aprender acerca de un comportamiento utilizando datos.

La manera en la cual aprenden de estos datos depende de el tipo de aprendizaje que se desee implementar.

Para mas informacion consultar el libro The Elements of Statistical Learning

Tipos de aprendizaje

  • Aprendizaje supervisado
  • Aprendizaje no supervisado
  • Aprendizaje por refuerzo
